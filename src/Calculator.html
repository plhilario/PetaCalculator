Other buttons:

Functional Features:

  1. Does not allow pressing of operators multiple times.
  2. Does not allow pressing of decimal point multiple times.
  3. Does not allow 15 characters in expression.
  4. Will not compute expression if the last character is an operator.
  5. Will not compute expression if the last character is a decimal point.
  6. Can use integers in the expression.
  7. Del button keeps track of the expression's state.
  8. Capable of handling memory.

Cosmetic Features:

  1. When cursor hovers over specific buttons, there will be a change in color.
  2. There are shadows over the buttons.
  3. Displays what is in memory.
  4. When resolution of the browser changes, the content of the html changes.
